## System requirements
* PHP 8.2
* Composer (global installed)
* PostgreSQL ^13 (14.9)

<br>

## Install
```sh
git clone git@github.com:Kolalexx/php-project-9.git
cd php-project-9
make install
```

## Start
```sh
make start
```

> For local start open localhost:8000 in your browser
